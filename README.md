# 🎓 EduVerse — Modern University Landing Page

A modern, fully responsive university landing page built with **Next.js**, **Tailwind CSS**, and **Framer Motion**.

This project was developed as part of an internship assignment to redesign and modernize the user experience of an educational institution website inspired by JG University.

---

## 🌐 Live Demo

🔗 Live Website:  
https://modern-university-landing.vercel.app/

---

## 📌 Project Overview

EduVerse is a visually appealing and performance-optimized landing page focused on delivering a modern educational website experience.

The project emphasizes:

- Clean UI/UX
- Fully responsive layouts
- Modern component architecture
- Interactive animations
- Reusable and scalable code structure
- Performance-first frontend development

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone YOUR_GITHUB_REPOSITORY_LINK
```

---

## 2️⃣ Navigate to Project

```bash
cd modern-university-landing
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Run Development Server

```bash
npm run dev
```

---

## 5️⃣ Open Browser

```bash
http://localhost:3000
```
